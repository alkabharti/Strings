### Solution : 

- Time Complexity : 


```java
String maxOdd(String s) {
    int i = 0;
    int j = s.length() - 1;
    while (i <= j) {
        if (s.charAt(j) % 2 != 0)
            return s;
        else {
            s = s.substring(0, s.length() - 1);
            j--;
        }
    }
    return s;
}

```
