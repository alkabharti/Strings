<h2> Roman Number to Integer : </h2>

Given a string in roman no format (s)  your task is to convert it to an integer . Various symbols and their values are given below. <br/>
I 1<br/>
V 5<br/>
X 10<br/>
L 50<br/>
C 100<br/>
D 500<br/>
M 1000<br/>

**Example 1:**

Input: <br/>
s = V<br/>
Output: 5 <br/>

**Example 2:**

Input: <br/>
s = III <br/>
Output: 3 <br/>
